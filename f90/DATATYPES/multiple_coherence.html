Statistical estimates for MT Transfer Functions

Multiple Coherencies: HPREDCOH and EPREDCOH


One can use an estimate of the impedance tensor to compute predicted E's, PEx and PEy , from the measured H's as:
PEx = Zxx Hx + Zxy Hy
PEy = Zyx Hx + Zyy Hy
In a similar manner, one can use an estimate of the admittance tensor to compute predicted H's, PHx and PHx from measured E's:
PHx = Yxx Ex + Yxy Ey
PHy = Yyx Ex + Yyx Ey
The E-predicted coherency EPREDCOH can be computed using the predicted E and the orthogonal measured H.
The H-predicted coherency HPREDCOH can be computed using the predicted H and measured E.
